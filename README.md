# MagicHelper
An optional assistant to Minecraft Magic servers or single

## Controls 
1. **Autoclicker**
   - Mouse button 1
     - On/Off - Mouse button 2
   - Custom button
     - On - Mouse button 2 + Right button
     - Off -  Mouse button 2
2. **Auto mining**
   - Horizontal mining
     - On - RControl + W
     - Off - Mouse button 2
   - Vertical mining
     - On - RControl + D
     - Off - Mouse button 2
   - Smart mining
     - On - RControl + S
     - Off - Mouse button 2
